Privacy Policy

WA Web Sidebar Toggle  ·  Last updated:

Short version: this extension collects no data whatsoever. Nothing leaves your browser. There are no servers, no analytics, no tracking.

What data does this extension collect?

None. WA Web Sidebar Toggle does not collect, transmit, store, or share any personal data, usage data, browsing history, or any other information.

What permissions does it use and why?

host_permissions for web.whatsapp.com — required to inject the toggle button into the WhatsApp Web page. The extension only runs on that domain and nowhere else.

storage — used exclusively to remember whether you have dismissed the "pin this extension" reminder. This value never leaves your browser and is not readable by any website or third party.

Does it access my WhatsApp messages?

No. The extension only manipulates the visibility of the sidebar panel (a layout element). It does not read, intercept, or process any messages, contacts, media, or any other WhatsApp content.

Does it communicate with any external servers?

No. The extension has no network requests of any kind. There is no backend, no telemetry endpoint, and no third-party SDK included.

Does it use cookies or local storage?

It uses sessionStorage on web.whatsapp.com only to remember the collapsed/expanded state of the sidebar for the current browser tab session. This is cleared automatically when the tab is closed. It also uses chrome.storage.local solely to remember whether the pin reminder has been dismissed.

Changes to this policy

If anything about the above ever changes, this page will be updated and the extension version will be bumped. You can always find the latest version at this URL.
